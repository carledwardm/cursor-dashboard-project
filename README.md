# Cursor Dashboard Demo 

This is a demo dashboard created to model Facility Maintenance software and showcase the technologies used. It's a single page application focused on design, layout and AI-assisted development. 

## Technologies
- Cursor AI (initial generation and refactors)
- React
- Next.js
- TypeScript

## Features
- Single page application
- Responsive design across screen sizes
- Modern dashboard layout
- Component-based architecture

## Installation

To run on your own machine:
    - git clone <your-repo-url>
    - cd <project-folder>
    - npm install
    - npm run dev

Then open:
    - http://localhost:3000
